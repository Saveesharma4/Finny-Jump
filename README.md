# Finny Jump

Finny Jump is a 2D game made in Unity inspired by Flappy Bird. The player controls a fish and tries to avoid obstacles while scoring as many points as possible.

## Features
- Endless gameplay
- Simple controls
- Score system
- Obstacle spawning

## Built With
- Unity
- C#
